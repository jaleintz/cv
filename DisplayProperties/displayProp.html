Display Properties Quick Notes

  • Block - "Display: Block;" for example
  • Inline
  • inline-Block
  • None

  • block elements take up the whole view port
  • -p- is a block element by default -p- can be change to inline by doing display: inline;
  • -span-img-a are inline display elements by default -span- can be change to block by doing display: block;
  • Block allows you to change the width but you can not have multiple elements on the same line Inline lets you have elements on the same line but you can not change the width Inline-Block lets you change the width and let you go on the same line as well -p-
